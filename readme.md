HTML es un **lenguaje marcado**; esto significa que cuenta con *reglas*, al ser marcado también se *diferencian los elementos* y posee la habilidad de *vincular* otras piezas de contenido ya sea dentro del mismo archivo, o un archivo distinto.

Es un lenguaje de **texto plano**, es decir *texto sin formato*.
Por lo tanto, para escribir HTML, debemos utilizar programas que se manejan únicamente en texto sin formato y no programas que guardan *metadatos de formato* junto con cada archivo.
